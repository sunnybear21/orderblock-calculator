# 📊 오더블록 계산기 (Order Block Calculator)

주식 종목의 **손절가, 익절구간, 진입구간**을 자동으로 계산해주는 웹 앱입니다.

## 🎯 기능

- 종목명 또는 종목코드로 검색
- 오더블록(세력 매집 흔적) 자동 감지
- 손절가 계산 (상승 OB 하단 기준)
- 진입 구간 표시 (상승 OB = 지지선)
- 익절 구간 표시 (하락 OB = 저항선)
- 매매 전략 자동 제안

## 📱 사용법

1. 종목명(예: 삼성전자) 또는 종목코드(예: 005930) 입력
2. "분석" 버튼 클릭
3. 결과 확인!

## 🔧 기술 스택

- Streamlit (웹 앱)
- 네이버 금융 크롤링 (API 키 불필요!)
- Pandas, NumPy

## 📝 오더블록이란?

오더블록은 세력(기관/큰손)이 대량 주문을 넣기 전 가격을 매집한 '발자국'입니다.

- **상승 오더블록**: 급등 직전 마지막 음봉 → 지지선 역할
- **하락 오더블록**: 급락 직전 마지막 양봉 → 저항선 역할

가격이 이 구간에 다시 오면 강력한 지지/저항이 발생합니다.

## ⚠️ 주의사항

- 투자 참고용이며, 투자 결정은 본인 책임입니다.
- 실시간 데이터가 아닌 네이버 금융 크롤링 데이터입니다.
